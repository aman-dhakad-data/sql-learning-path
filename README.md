# SQL Learning Path

A structured SQL learning repository documenting my SQL learning journey from fundamentals to intermediate and advanced querying concepts.

This repository contains 21 practice files covering core SQL concepts through hands-on exercises, business-style scenarios, data cleaning tasks, analytical queries, and interview-oriented challenges.

## Topics Covered

### SQL Fundamentals

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* DISTINCT

### Database Objects

* DDL (CREATE, ALTER, DROP)
* DML Operations
* Constraints

### Data Analysis & Querying

* Joins
* Advanced Joins
* Multi-Table Analysis
* Set Operators
* Aggregations
* Window Functions
* Subqueries
* Common Table Expressions (CTEs)

### Data Cleaning & Transformation

* String Functions
* Date & Time Functions
* NULL Handling
* CASE Statements
* Data Standardization
